# 🛒 SwiftKart Business Analytics Dashboard

An end-to-end **Business Analytics** project built using **SQL** and **Power BI** to analyze retail business performance, customer behavior, sales trends, and operational efficiency.

This project demonstrates SQL querying, data modeling, KPI development, and interactive dashboard design to support business decision-making.

---

# 📌 Project Overview

SwiftKart is a fictional retail grocery platform. This project focuses on transforming raw transactional data into meaningful business insights using SQL and Power BI.

The project covers:

- Sales Performance Analysis
- Customer Analytics
- Delivery & Operations Analysis
- Interactive Power BI Dashboard
- SQL Business Queries
- Business Performance KPIs
